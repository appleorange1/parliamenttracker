About

The Victorian Parliament Tracker generates reports for divisions that happen in Parliament. This page attempts to explain in further detail how it works, what it can and can't do and how you can help.

How is the data on this website generated?

Most of the pages on this website are made with the help of an open-source program which extracts data from the Votes and Proceedings files (for the Legislative Assembly) and the Minutes of Proceedings files (for the Legislative Council).

The human-readable summaries are written by humans.

How can I help?
  • Write more human-readable summaries of divisions and send them to me
  • Make improvements, extensions or fixes to the program or website
  • Spread the word!
I can't find a certain issue on here!

There are several possible reasons for this:

  • This website can only provide voting results on issues where a division was held. In many cases, only a voice vote is held, which means that the votes of individual MPs aren't recorded.

  • The Legislative Council data goes back to 3 September 2015 and the Legislative Assembly data goes back to 10 February 2015.

    We have no objection to including older data; it just isn't a priority for us at the moment. If you can get the program working with older data, let us know and we'll include it. If you need help understanding where to start, just ask and we'll try to be of assistance.

  • There may be a bug in the program which skips over your division. If this is the case, send us a link to the Votes and Proceedings/Minutes of Proceedings file with the division in it.
Are there any other websites like this?

The OpenAustralia Foundataion runs a website called They Vote For You, which tracks members of the Federal Parliament.

The computer program which generates these pages takes the title of each section and produces a summary of how members voted in that division.

In many cases, the section titles are misleading. This is because the actual question on which the house divides is not picked up by the computer program.

For example, instead of actually voting on the bill, the house may actually be voting on whether an amendment to the bill should be accepted or whether the debate should be adjourned to a later time.
